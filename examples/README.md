# Exemplos

Scripts de exemplo de uso da biblioteca. Cada exemplo deve:

- usar **apenas dados fictícios** (CPF `111.111.111-11`, processo `00000.000000/0000-00`);
- ser executável de forma isolada;
- não conter credenciais nem caminhos absolutos da máquina do autor.
